# FindHub - Campus Marketplace

FindHub is a comprehensive marketplace platform designed specifically for campus communities, enabling students to buy, sell, and trade items within their university ecosystem.

## Features

- User authentication with Google login
- Product listing and browsing
- Shopping cart functionality
- Responsive design for all devices
- Firebase backend integration

## Tech Stack

- React 19 with Vite
- Firebase Authentication and Firestore
- React Router for navigation
- Lucide React for icons

## Installation

1. Clone the repository
2. Install dependencies with `npm install`
3. Set up Firebase configuration in `.env` file
4. Run with `npm run dev`
